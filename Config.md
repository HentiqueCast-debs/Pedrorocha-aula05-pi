# criação do arquivo  de configuração

## Passo a passo(para Windows):

* 1 Primeiro abra o terminal de comando do Windows escolha um ponto de referencia para a criação da pasta e execute o seguinte comando:

`` mkdir django_aulas`` 

  `` cd django_aulas``

   mkdir:(Make Directory) cria uma pasta no seu computador 
   cd:(Change Dictory) enta na pasta que acabamos de criar 

 * 2 Vamos criar o ambiente virtual venv para configurar e estruturar nosso projeto, execute no terminal :

`` python -m venv venv ``


 ``venv\Script\activated``

 * 3 instale o framework django via terminal executando :

 `` pip install django ``
 
 ``Collecting django``

para verificar a versao instalada ue :

``django-admin --version``

* 4 Para criar a estrutura do projeto usando as ferramentas do django execute no terminal :

``django-admin startproject meu_projeto``

e para usar o projeto use:

``meu_projeto``

* 5 Para usar o vscode como ferramenta de visualização e edição do projeto use 
  
`` code .``
